配送范围及时间

配送范围及时间

配送方式：

纳车品与两家快递公司合作配送商品，进货商在纳车品成功购买商品的每一个订单（限一个邮寄地址），系统都会默认生成一个包裹。纳车品目前暂时不提供自选物流的服务，我们会根据订单中的收货地址和商品种类选择物流公司为会员配送。。

配送范围：

纳车品配送范围覆盖全国大部分地区（港澳台地区除外）。

配送费用：

纳车品快递运费以页面提示为准，个别特殊商品可能需要增加运费，具体会在该商品介绍页注明。

发货时间：

会员下单成功后，相关专业人员在12个小时内对会员所订购的商品、邮寄地址、款项支付等信息进行审核。审核通过后，一般在1-2个工作日内将包裹发出；包裹发出后，系统将配送相关信息更新至进货商的帐户信息中，进货商可通过后台的“订单处理－订单列表”对应订单后点击“查看”查询包裹的派送情况。

注：部分特殊商品发货时间需以商品详情页为准，敬请谅解！

配送追踪：

如进货商已成功订购纳车品供销平台的商品，可以登陆后台查看订单状态，在订单状态中查询到快递单号，通过快递官网，输入配送物流单号，便可查询到商品的配送详情。
