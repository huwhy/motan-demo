服务保障

8天无理由放心退

保障范围：

进货商提出退款申请所指向的商品在“8天无理由退换货”服务保障范围内（部分商品及类目除外）。

申请时间：

进货商需在确认签收商品之日起8天内（按照物流签收后的第二天零时起计算时间，满192小时为8天）提出申请。

维权方式：

在满足8天无理由退换货申请条件的前提下，于纳车品供销平台后台提出“8天无理由退换货”申请。

在收到进货商“8天无理由退换货”赔付申请后，纳车品供销平台有权根据协调情况要求进货商提供必要证明，并确认及判定。

运费说明：

纳车品支持提供无理由退货服务的商品，进货商以无理由形式退货或拒签的，来回运费均由进货商承担。

如果是商品质量问题，收到的货物与网上描述不符而导致的退货，退货费用则由纳车品供销平台承担。

其他说明：

申请金额仅以买家实际支付的商品价款为限。

买家的赔付申请在形式上符合相关法律法规的规定。

退换货要求具备商品收到时完整的外包装、配件、吊牌等；购买物品被洗过、用过、人为破坏或标牌拆卸的不予退换；所有预定或订制特殊尺码的不予退换。

特别提醒：

为避免由于商品滞留造成的经济损失，所有退换货商品，进货商应在7天内发回（以物流签收运单显示时间为准）超过规定时间仍不能将退换商品发出的，不再支持进货商退换货申请。
